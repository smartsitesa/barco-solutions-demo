What the backend includes

Google Authentication
Admin users sign in with Google before changing bot settings.
Bot Parameters
Keywords, fixed replies, blocked topics, fallback text and WhatsApp number.
No AI API needed
The bot greets, matches keywords/parameters, replies from configured text, and hands off when unsure.
WhatsApp Handoff
When no parameter matches, the bot asks if the visitor wants an agent and gives a WhatsApp link.

Deployment note

GitHub Pages cannot run a Node backend. To make the parameter bot fully live, host the backend on a server/subdomain, then embed:

<script src="https://api.example.com/widget.js" data-bot-id="barco-demo"></script>
